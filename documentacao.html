Acessibilidade

Acessibilidade é a possibilidade de acessar um lugar, serviço, produto ou informação de maneira segura e autônoma, sem nenhum tipo de barreira, beneficiando a todas as pessoas, com ou sem deficiência, em todas as fases da vida.

Como que funciona a nossa ferramenta?

É simples! Nessa página iremos explicar como! Sinta-se livre para usar ela em seu site.

Este é o código de JavaScript que nós usamos para aumentar a fonte dos textos:

function funcao(){
var x = document.getElementsByClassName("Nome da classe dos textos");
for ( var i = 0; i < x.length; i++) {
x[i].style.fontSize="1.5rem";
}
};

E este é o código para diminuir a fonte dos textos:

function funcao(){
var x = document.getElementsByClassName("Nome da classe dos textos");
for ( var i = 0; i < x.length; i++) {
x[i].style.fontSize=" ";
}
};

No código para diminuir, o local que fica o tamanho da fonte fica vazio, isso faz que o texto volte para o tamanho original dele.
